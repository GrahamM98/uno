run the uno bash script to start playing
when run, you are prompted to choose a game mode.

typing pvp will activate the multiplayer mode
typing pvc will activate the singleplayer mode

to play, type the name of a card in your hand that matches either the color or number/type of the card

the color of the card is denoted by the first character

wild cards don't have a character denoting color at the beginning(wild and +4)
